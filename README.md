# AStar Product Recommendation System

A Python implementation of a simplified product recommendation system using the **A\*** (A-Star) search algorithm. Products are represented as nodes in a fully connected graph, and recommendations are generated by finding the optimal path between products using a heuristic based on conversion probability.

## Features

- Product representation with:
  - Name
  - Category
  - Conversion probability
- Fully connected product graph
- A* pathfinding algorithm
- Custom heuristic that prioritizes products with higher conversion probability
- Simple and easy-to-understand implementation for educational purposes

## How It Works

The recommendation system consists of two main classes:

### Product

Represents a product with:

- Name
- Category
- Conversion probability

### AStarRecommendation

Responsible for:

- Building the product graph
- Executing the A* algorithm
- Finding the best recommendation path between two products

The heuristic function favors products with higher conversion probabilities by returning the negative value of the probability, allowing A* to prioritize more attractive recommendations.

## Algorithm

The implementation uses the **A\*** search algorithm.

For each node:

- **g(n)**: accumulated cost from the start product.
- **h(n)**: heuristic based on conversion probability.
- **f(n) = g(n) + h(n)**.

Products with higher conversion probabilities receive better priority during the search.
